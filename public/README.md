# Public Assets

This folder contains static assets for the application.

## Required Assets

### hamburger.gif

You need to add the `hamburger.gif` loading animation file to this directory.

This file can be obtained from the original frontend repository:
https://github.com/mahanmashoof/recipe-generator-fe/blob/main/public/hamburger.gif

Or you can use any other loading GIF animation of your choice.
